# Basic-Image-Classifier

## Dependencies

1. Get project dependencies 
```
conda env update -f environment.yml
```

2. When prompted, activate virtual environment
```
conda activate flask-basic
```

## Run Flask

/path/to/basic-image-classifier>
```
$ python app.py
```
